
# devtools 开发工具

```bash
# 安装、更新环境
pub global activate devtools # If you have `pub` on your path.
flutter pub global activate devtools # If you have `flutter` on your path.

# 启动服务
pub global run devtools   # If you have `pub` on your path.
flutter pub global run devtools   # If you have `flutter` on your path.
```
