
`Flutter`几乎所有的东西都是`Widgtes`，包括布局都是`Widgtes`。

布局组件的`child`有两种形式:

* Single-child layout widgets -> `Widget child`
* Multi-child layout widgets -> `List<Widget> children = const <Widget>[]`
